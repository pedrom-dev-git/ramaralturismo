# R. Amaral — Turismo e Transporte

Website da empresa R. Amaral, especializada em transporte escolar e turismo/viagens em grupo.

## Design

- **Figma Prototype:** [Abrir no Figma](https://www.figma.com/proto/pxCxtAsVGsQj5MxlghwW75/R-amaral?node-id=45-4946)
- **Resolução:** 1440px (Desktop)
- **Documentação de Assets:** [ASSETS.md](./ASSETS.md)

## Seções do Site

1. **Header** — Navbar com logo R. Amaral + menu de navegação
2. **Hero** — Imagem de van + formulário de orçamento via WhatsApp
3. **Benefícios** — 4 cards (Segurança, Pontualidade, Conforto, Atendimento)
4. **Parceiros** — Carousel de logos
5. **Serviços** — Transporte Escolar + Turismo e Viagens
6. **Feature News** — Carousel de notícias/destinos
